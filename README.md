# 🛡️ Study Guard Pro - Smart Lockdown
**اسٹڈی گارڈ پرو: بچوں کی تعلیم اور وقت کی حفاظت کے لیے ایک اسمارٹ ٹول**

یہ ایک کروم ایکسٹینشن (Chrome Extension) ہے جو خاص طور پر والدین کے لیے بنائی گئی ہے تاکہ وہ بچوں کے انٹرنیٹ استعمال کو تعلیمی اوقات میں بہتر طریقے سے کنٹرول کر سکیں۔

---

### ✨ نمایاں خصوصیات (Key Features)

* **⏲️ اسمارٹ ٹائمنگ (9 AM - 5 PM):** مخصوص ویب سائٹس (YouTube, Facebook, TikTok) کو تعلیمی اوقات میں خودکار بلاک کرتی ہے۔
* **🔑 پیرنٹل کنٹرول (Daily Limit):** بچہ صرف والدین کے پاس ورڈ سے دن میں ایک بار 10 منٹ کی عارضی اجازت لے سکتا ہے۔
* **🌟 موٹیویشنل کوٹس:** بلاک شدہ پیج پر بچوں کی ہمت بڑھانے کے لیے خوبصورت اردو اقوال (Quotes) نظر آتے ہیں۔
* **⚡ ماسٹر کنٹرول (Master Bypass):** والدین کے پاس اپنا ایک مخصوص "ماسٹر پاس ورڈ" ہے جس سے وہ ضرورت پڑنے پر پورے دن کے لیے ایکسٹینشن آف کر سکتے ہیں۔
* **📊 الرٹ سسٹم:** گوگل فارم (Google Form) کے ذریعے والدین کو بلاک شدہ سائٹ کھولنے کی کوشش پر خاموشی سے اطلاع مل جاتی ہے۔

---

### 🛠️ ٹیکنالوجی (Technologies Used)

* **Language:** JavaScript (Chrome Manifest V3)
* **Frontend:** HTML5, CSS3 (Minimalist & Pastel UI)
* **Storage:** Chrome Storage API (for session management)
* **Communication:** Background Service Workers & Content Scripts

---

### 🚀 انسٹال کرنے کا طریقہ (How to Install)

1. اس ریپوزٹری کو **Download ZIP** کریں یا **Clone** کریں۔
2. اپنے گوگل کروم براؤزر میں `chrome://extensions/` پر جائیں۔
3. اوپر دائیں کونے سے **Developer Mode** آن کریں۔
4. **Load Unpacked** بٹن پر کلک کریں اور اس فولڈر کو سلیکٹ کریں۔
5. ایکسٹینشن اب فعال (Active) ہے!
